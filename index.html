Ques8. What is box sizing?

Ans.
-Box sizing it is used to change height and width of element.
-And also there is property of box sizing that is
(1)content-box:-in this height, width, min, max properties are included only in content.borer and padding are not included.
(2)border-box:-in this it includes width and height properties and also include content padding and boder.
-Width and height of an element is calculated like this:
width + padding + border = actual width of an element
height + padding + border = actual height of an element


Ques9. What is animation delay?

Ans.
-This property tells us about the delay in the start of an animation.
animation-delay value is defines is milliseconds(ms).
This allows an animation to begin execution some time after it is applied.


Ques10. Which property you will use to merge cells vertically in grid?

Ans.
-grid-column-start and grid-column-end property can be used to merge cells vertically.
- It also has a shorthand i.e., grid-column: start/end.
- If you want to merge two elements in row then we will have to use grid-row property inside the child element.
- These properties are applicable only to child elements not to parent elements


Ques11. Which property you will use to merge cells horizontally in grid?

Ans.
-grid-row-start and grid-row-end property can be used to merge cells horizontally.
- It also has a shorthand i.e., grid-row: grid-row-start/grid-row-end;
